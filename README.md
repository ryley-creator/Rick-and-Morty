# Rick and Morty Flutter App

Приложение на **Flutter** для просмотра персонажей из Rick and Morty с возможностью добавления в избранное. Используется локальное кэширование через **Hive**, поддерживается пагинация и сортировка избранного.

---

## Функционал
- Просмотр персонажей с API Rick and Morty
- Добавление/удаление персонажей в избранное
- Локальное кэширование страниц персонажей через Hive
- Пагинация и загрузка следующих страниц
- Сортировка избранного по ID (по умолчанию) и по имени
- Placeholder для изображений и кэширование через `cached_network_image`
- Интерактивная кнопка избранного с обновлением UI
- Переключение темы (Light/Dark Theme) с сохранением выбора пользователя
---

## Инструкция по сборке и запуску

1. Клонировать проект:
```bash
git clone <your-repo-url>
```
2. Перейти в папку проекта:
```bash
cd <project-folder>
```
3. Установить зависимости:
```bash
flutter pub get
```
4. Запустить на эмуляторе или реальном устройстве:
```bash
flutter run
```
5. Важно: Для работы с Hive необходимо инициализировать его в main.dart перед запуском приложения:
```bash
await Hive.initFlutter();
```
